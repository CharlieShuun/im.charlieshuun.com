# 個人作品集與技術實作網站
>  [im.charlieshuun.com](https://im.charlieshuun.com)

專案概述
本專案為我的個人作品集，展示了我在高中期間對 **AI 實作**與**網路安全**的探索過程。網站基於 HTML5 UP 模板進行深度改寫，並透過自建伺服器環境進行部署。

部署與資安實作 (Deployment & Security)
雖然本網站為靜態網頁，但我將其部署於 **QNAP NAS 原生伺服器環境**，並實施了以下安全策略：
1. **反向代理 (Reverse Proxy)：** 隱藏內網實體 IP，增加一道外部防禦。
2. **HTTPS 加密：** 導入 SSL 憑證，確保使用者連線之資料傳輸安全。
3. **主動防禦：** 透過 QuFirewall 監控流量，並針對日誌中偵測到的 30,000+ 次掃描嘗試，實施 IP 封鎖。

核心檔案說明
* `index.html`: 網站主體，包含我對 MNIST AI 實作與硬體維修的經驗總結。
* `assets/`: 自定義的樣式與多媒體資源。
